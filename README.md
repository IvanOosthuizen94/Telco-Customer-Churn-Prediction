# Telco-Customer-Churn-Prediction
A machine learning classification project predicting telecommunications customer churn. I trained Logistic Regression and Random Forest models, evaluating performance via confusion matrices, precision, and recall to recommend the most effective model for targeted customer retention campaigns.
----------------------------------------------------------------------------
1. Navigate to the directory  
2. Launch Jupyter Notebook  
3. Open the Notebook: In your browser, click on telco_customer_churn.ipynb to view the code, run the models, and explore the comparative evaluation metrics.  

### Project Structure
1. telco_customer_churn.ipynb: The main Jupyter Notebook containing the Python code, model training, and performance evaluation.  
2. Telco-Customer-Churn.csv: The dataset containing customer account information, service details, and churn status.
